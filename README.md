porID Generator

أداة تشتغل على Termux لتوليد بيانات وهمية + صورة شخصية + رمز QR + تخزين كل شي بملف خاص.

## مميزات الأداة:
- توليد اسم، هوية، عنوان، إيميل، رقم
- توليد دولة وبصمة وتوقيع وهمي
- صورة شخصية وهمية من AI
- رمز QR لكل هوية
- تخزين كل هوية داخل مجلد
